# 🏥 Self Help Medicare - Healthcare App
## 📱 About the Project
Self Help Medicare is an Android application designed to provide information about common diseases, symptoms, precautions, and home remedies. The app aims to promote safe self-care practices and improve healthcare awareness.
## 🚀 Features
* Information about common diseases
* Symptoms and precautions
* Home remedies guidance
* Multilingual support
* Simple and user-friendly interface
## 🛠️ Tech Stack
* Kotlin
* XML (Android UI)
* Firebase (Basic)
* Android Studio
## 🏆 Achievement
* Presented in **Avishkar Research & Innovation Competition**
